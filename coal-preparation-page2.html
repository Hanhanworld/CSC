SIWAREX卡片設定、校正程式

SFB53功能說明

參數 資料格式 說明
EN BOOL 啟用SFB53模組
REQ BOOL 執行寫入命令
ID DWORD SIWAREX卡片起始位置
INDEX INT SIWAREX DR功能數字
LEN INT 欲寫入的資料長度
RECORD ANY TYPE 從PLC位置寫入SIWAREX卡片
DONE BOOL 寫入指令完成

SFB53運行原理總結

當 REQ 被觸發時，WRREC 開始將 RECORD 參數中指定的資料寫入到目標裝置 (ID) 中對應的索引 (INDEX) 下的記錄。

如果操作成功，則 DONE 置為 TRUE，且 LEN 參數傳回實際寫入的資料長度。如果操作失敗，ERROR 設為 TRUE，且 STATUS 參數會傳回特定的錯誤代碼。

WRREC 常用於將參數、配置或其他資料傳送到裝置或模組中，以實現資料寫入或裝置配置更新。

SFB52功能說明

參數 資料格式 說明
EN BOOL 啟用SFB52模組
REQ BOOL 執行讀取命令
ID DWORD SIWAREX卡片起始位置
INDEX INT SIWAREX DR功能數字
LEN INT 欲讀取的資料長度
RECORD ANY TYPE 從SIWAREX卡片讀取放入PLC位置
VALID BOOL 讀取成功=1
BUSY BOOL 讀取中=1
ERROR BOOL 讀取錯誤
STATUS DWARD 讀取錯誤時的錯誤代碼
LEN INT 讀取成功時將顯示資料長度

SFB52運行原理總結

當 REQ 被觸發時，RDREC 開始讀取指定裝置 (ID) 中對應索引 (INDEX) 的資料記錄，並將資料存放到 RECORD 參數指定的儲存區域內。

如果操作成功，則 VALID 設為 TRUE，且 LEN 參數傳回實際讀取的資料長度。如果操作失敗，ERROR 設為 TRUE，且 STATUS 參數會傳回特定的錯誤代碼。

這些參數可讓 RDREC 讀取來自不同裝置的數據，適用於多種資料擷取或診斷情境。
